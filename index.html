Chapter 1 Probability basics

1.1 Probability spaces and a bit of measure theory

Underlying any probabilistic model is a probability space (Ω,ℱ,ℙ), where

  • •

    Ω is the sample space that contains all possible outcomes ω, called elementary events;

  • •

    the event space ℱ⊂2Ω is a collection of subsets of the sample space that we wish to assign a probability to. A set A∈ℱ is called event;

  • •

    ℙ is a probability measure that assigns to each event A∈ℱ a probability ℙ⁢(A)∈[0,1].

Example 1.1.1.

Consider the experiment of throwing a fair coin twice. We model Ω={H⁢H,H⁢T,T⁢H,T⁢T}={ωi:i=1,…,4} and

ℱ=2Ω={ ∅,Ω,{H⁢H},{H⁢T},{T⁢H},{T⁢T},{H⁢H,H⁢T},{H⁢H,T⁢H},{H⁢H,T⁢T},{H⁢T,T⁢H},{H⁢T,T⁢T},
{TH,TT},{HH,HT,TH},{HH,HT,TT},{HH,TH,TT},{HT,TH,TT}}

Since the coin is fair, each of the 4 elementary events {H⁢H},{H⁢T},{T⁢H},{T⁢T} has the same probability, which is given by 1/4, i.e., ℙ⁢({ω})=1/4 for all ω∈Ω. The probability that one of the distinct outcomes H⁢T and T⁢H happens is intuitively obtained by summing up the probabilities, i.e., ℙ⁢({H⁢T,T⁢H})=ℙ⁢({H⁢T})+ℙ⁢({T⁢H})=1/4+1/4=1/2. More generally, for C⊂{1,…,n}, we may set

ℙ⁢({ωi:i∈C})=ℙ⁢(⋃i∈C{ωi})=∑i∈Cℙ⁢({ωi})=|C|/4.

Note however that with this definition, probabilities of unions of events (translates to “at least one of the events happens”) are not always obtained by summing up:

ℙ⁢({H⁢T}∪{H⁢T,T⁢H})=ℙ⁢({H⁢T,T⁢H})=1/2≠3/4=ℙ⁢({H⁢T})+ℙ⁢({H⁢T,T⁢H}).

The reason here: the events are not disjoint! With the above model we can for example identify “heads appears in first throw” with the event {H⁢H,H⁢T}, which has probability 1/2, or “heads appears at least once” with the event {H⁢T,T⁢H,H⁢H}, which has probability 3/4. A reformulation of this would be “tails does not appear twice” and the probability can be expressed as

3/4=1−1/4=1−ℙ⁢(“tails appears twice”)=1−ℙ⁢({T⁢T})=1−ℙ⁢(Ω∖{H⁢T,T⁢H,H⁢H}).

To make these notions mathematically rigorous, we introduce some fundamental concepts from measure theory, where one builds a consistent theory of assigning volumes to sets on general spaces. First we deal with the types of event spaces that we allow for a probability space.

Definition 1.1.2.

ℱ⊂2Ω is called a σ-algebra if all of the following are satisfied:

  1. (i)

    Ω∈ℱ

  2. (ii)

    A∈ℱ⟹Ac≔Ω∖A∈ℱ

  3. (iii)

    (An)n∈ℕ⊂ℱ⟹⋃n∈ℕAn∈ℱ

If ℱ is a σ-algebra over Ω, we call (Ω,ℱ) a measurable space.

Lemma 1.1.3.

Any σ-algebra ℱ has the following properties:

  1. (i)

    ∅∈ℱ

  2. (ii)

    A,B∈ℱ⟹A∪B∈ℱ

  3. (iii)

    A,B∈ℱ⟹A∩B∈ℱ

  4. (iv)

    (An)n∈ℕ⊂ℱ⟹⋂n∈ℕAn∈ℱ

  5. (v)

    A,B∈ℱ⟹A∖B∈ℱ

Proof.

Exercise. ∎

Example 1.1.4.

The simplest examples of σ-algebras are the trivial σ-algebra ℱ={∅,Ω} and the power set ℱ=2Ω (sometimes called discrete σ-algebra).

For uncountable spaces Ω, the power set 2Ω is usually a poor choice because it is simply too big in the sense that we cannot design meaningful measures that consistently assign volumes to all subsets of Ω (look up the Banach–Tarski paradox if you are interested why). Instead, we usually start with a nice family of subsets ℰ of Ω and works with the smallest σ-algebra that contains ℰ.

Definition 1.1.5.

Let ℰ be a family of subsets of Ω. Then σ⁢(ℰ) is defined as the smallest σ-algebra over Ω that contains ℰ, or equivalently,

σ⁢(ℰ)≔⋂ℱ∈𝒜ℱ,

where 𝒜={ℱ⊂2Ω:ℱ⁢ is a ⁢σ⁢-algebra and ⁢ℰ⊂ℱ}. The family ℰ is called a generator of σ⁢(ℰ).

Exercise 1.1.6.

Let Ω be a countable set and ℰ={{ω}:ω∈Ω}. Show that σ⁢(ℰ)=2ℰ.

For our statistical purposes the by far most important σ-algebra is the Borel σ-algebra ℬ⁢(ℝ) over the real numbers ℝ, which is defined as

ℬ⁢(ℝ)≔σ⁢({O⊂ℝ:O⁢ open}),

and we will always implicitly equip ℝ with this σ-algebra when considering it as a measurable space. The Borel σ-algebra has the following simpler characterisation.

Lemma 1.1.7.

All of the following families of sets are generators of ℬ⁢(ℝ):

  1. (i)

    ℰ1={(a,b):−∞<a≤b<∞}

  2. (ii)

    ℰ2={[a,b]:−∞<a≤b<∞}

  3. (iii)

    ℰ3={(a,b]:−∞<a≤b<∞}

  4. (iv)

    ℰ4={(−∞,a):a∈ℝ}

  5. (v)

    ℰ5={(−∞,a]:a∈ℝ}

Proof.

We only prove σ⁢(ℰ1)=ℬ⁢(ℝ), the remaining statements are left as an exercise. Since any open interval (a,b) is open in ℝ we have ℰ1⊂{O⊂ℝ:O⁢ open} and therefore

σ⁢(ℰ1)⊂σ⁢({O⊂ℝ:O⁢ open})=ℬ⁢(ℝ). (1.1)

Conversely, let O⊂ℝ be an arbitrary open set. Recall that O can then be written as a countable union of open intervals, that is, we can find sequences (an)n∈ℕ,(bn)n∈ℕ such that O=⋃n∈ℕ(an,bn). Since (an,bn)∈ℰ1⊂σ⁢(ℰ1) for all n, property (iii) of a σ-algebra implies that

O=⋃n∈ℕ(an,bn)⏟∈σ⁢(ℰ1)∈σ⁢(ℰ1).

Since O was chosen arbitrarily we conclude that

{O⊂ℝ:O⁢ open}⊂σ⁢(ℰ1),

and therefore also

ℬ⁢(ℝ)=σ⁢({O⊂ℝ:O⁢ open})⊂σ⁢(ℰ1). (1.2)

Combining (1.1) and (1.2) gives the result. ∎

Now that we have understood what a σ-algebra is, let us move on to the notion of a probability measure. We start a bit more general.

Definition 1.1.8.

A measure μ on a measurable space (Ω,ℱ) is a function μ:ℱ→[0,∞] that satisfies the following properties:

  1. (i)

    μ⁢(∅)=0

  2. (ii)

    if (An)n∈ℕ is a pairwise disjoint sequence in ℱ, it holds that

    μ⁢(⋃n∈ℕAn)=∑n∈ℕμ⁢(An) (σ-additivity)

The measure μ is called

  1. (i’)

    finite, if μ⁢(Ω)<∞

  2. (ii’)

    σ-finite, if there is a sequence (An)n∈ℕ in ℱ such that Ω=⋃n∈ℕAn and μ⁢(An)<∞ for all n.

Remark 1.1.9.

Note that by taking ∅=An+1=An+2=⋯ it follows from σ-additivity that for any finite, mutually disjoint collection A1,…,An∈ℱ we have μ⁢(A1∪⋯∪An)=∑k=1nμ⁢(Ak).

Definition 1.1.10.

A finite measure ℙ on a measurable space (Ω,ℱ,ℙ) is called probability measure if ℙ⁢(Ω)=1. The triplet (Ω,ℱ,ℙ) is called probability space.

Definition 1.1.11.

A probability distribution (or simply distribution) is a probability measure on (ℝ,ℬ⁢(ℝ)).

Lemma 1.1.12.

Let (Ω,ℱ,ℙ) be a probability space and A,B∈ℱ be arbitrary events. Then,

  1. (i)

    if A⊂B, it holds that ℙ⁢(B∖A)=ℙ⁢(B)−ℙ⁢(A). In particular, ℙ⁢(Ac)=1−ℙ⁢(A).

  2. (ii)

    ℙ⁢(A∪B)=ℙ⁢(A)+ℙ⁢(B)−ℙ⁢(A∩B). In particular, ℙ⁢(A∪B)≤ℙ⁢(A)+ℙ⁢(B).

  3. (iii)

    if A⊂B, it holds that ℙ⁢(A)≤ℙ⁢(B).

Remark 1.1.13.

Analogous statements can be derived whenever we deal more generally with a finite measure μ. For infinite measures we need to be a bit more careful. E.g., while it is always true that μ⁢(A∪B)≤μ⁢(A)+μ⁢(B), we can only make sense of μ⁢(A)+μ⁢(B)−μ⁢(A∩B) if μ⁢(A∩B)<∞, in which case indeed also μ⁢(A∪B)=μ⁢(A)+μ⁢(B)−μ⁢(A∩B).

Proof.
  1. (i)

    A⊂B implies B=A⊎(B∖A), whence by σ-additivity, ℙ⁢(B)=ℙ⁢(B∖A)+ℙ⁢(A), or equivalently, ℙ⁢(B∖A)=ℙ⁢(B)−ℙ⁢(A). Letting B=Ω, we obtain from this

    ℙ⁢(Ac)=ℙ⁢(Ω∖A)=ℙ⁢(Ω)−ℙ⁢(A)=1−ℙ⁢(A).
  2. (ii)

    Let C=A∩B. We can write A∪B=(A∖C)⊎(B∖C)⊎C, which gives (using C⊂A,C⊂B and σ-additivity)

    ℙ⁢(A∪B)=ℙ⁢(A∖C)+ℙ⁢(B∖C)+ℙ⁢(C) =(i)⁢ℙ⁢(A)−ℙ⁢(C)+ℙ⁢(B)−ℙ⁢(C)+ℙ⁢(C)
    =ℙ⁢(A)+ℙ⁢(B)−ℙ⁢(C)
    =ℙ⁢(A)+ℙ⁢(B)−ℙ⁢(A∩B).
  3. (iii)

    immediate from (i) since ℙ⁢(B∖A)≥0.

∎

Lemma 1.1.14 (Continuity of measures).

Let (Ω,ℱ,μ) be a measure space.

  1. (i)

    If (An)n∈ℕ⊂ℱ is an increasing sequence in the sense that An⊂An+1 for all n∈ℕ, then

    μ⁢(⋃n∈ℕAn)=limn→∞μ⁢(An).
  2. (ii)

    If (An)n∈ℕ⊂ℱ is a decreasing sequence in the sense that An⊃An+1 for all n∈ℕ, and moreover μ⁢(A1)<∞, then

    μ⁢(⋂n∈ℕAn)=limn→∞μ⁢(An).
Proof.

See Appendix A. ∎

Of special interest to us will be the following measures on (ℝ,ℬ⁢(ℝ)), which are in a certain sense on opposite ends of the spectrum. The Lebesgue measure λ is the unique measure that satisfies

λ⁢((a,b])=b−a,−∞<a≤b<∞,

and therefore assigns a volume to any set in ℬ⁢(ℝ) that we would consider as natural based on our physical perception of the world. The Dirac measure δx at a point x∈ℝ is defined as

δx⁢(A)={1,x∈A0,x∉A,

i.e., it is a point mass that assigns unit volume to a measurable set A iff x∈A. In particular we have δx⁢({x})=1=δx⁢(ℝ), so that δx is our first example of a probability measure. Conversely to this, for the Lebesgue measure it holds

λ⁢({x})=λ⁢(⋂n∈ℕ(x−1/n,x+1/n])=limn→∞λ⁢((x−1/n,x+1/n])=limn→∞2n=0.

For the the second equality we used continuity of measures from Lemma 1.1.14. This means that any singleton {x} is a nullset for the Lebesgue measure according to the next definition.

Definition 1.1.15.

Let μ be a measure on some measurable space (Ω,ℱ). A set N∈ℱ such that μ⁢(N)=0 is called a μ-nullset. If some property holds for all ω∈Ω∖N, where N is a μ-nullset, we say that this property holds μ-almost everywhere (μ-a.e.). If μ is a probability measure we say instead μ-almost surely (μ-a.s.).

Exercise 1.1.16.

Show the following:

  1. (i)

    If ℙ1,ℙ2 are probability measures and λ1,λ2≥0, then λ1⁢ℙ1+λ2⁢ℙ2 is a probability measure iff λ1+λ2=1.

  2. (ii)

    For any a,b∈ℝ such that a<b it holds that

    λ⁢([a,b])=λ⁢([a,b))=λ⁢((a,b])=λ⁢((a,b)).
  3. (iii)

    The Lebesgue measure λ is σ-finite but not finite.

1.2 Independence and Conditional Probability

Throughout this section we fix a probability space (Ω,ℱ,ℙ).

Definition 1.2.1 (Independence).

Two events A,B∈ℱ are defined to be independent if

ℙ⁢(A∩B)=ℙ⁢(A)⁢ℙ⁢(B).
Example 1.2.2.

Let the probability space from Example 1.1.1 be given. Consider the events A={H⁢T,H⁢H}≜“heads appears in first throw” and B={H⁢T,T⁢T}≜“tails appears in second throw”. Then

ℙ⁢(A∩B)={H⁢T}=1/4=12⋅12=ℙ⁢({H⁢T,H⁢H})⁢ℙ⁢({H⁢T,T⁢T})=ℙ⁢(A)⁢ℙ⁢(B),

so A and B are independent.

Definition 1.2.3 (Conditional Probability).

Let A∈ℱ s.t. ℙ⁢(A)>0. The conditional probability ℙ(⋅∣A) on (Ω,ℱ) is defined by

ℙ⁢(B∣A)=ℙ⁢(A∩B)ℙ⁢(A),∀B∈ℱ,

and we call ℙ⁢(B∣A) the conditional probability of B given A.

The definition entails that if ℙ⁢(A)>0, an event B∈ℱ is independent of A, iff ℙ⁢(B∣A)=ℙ⁢(B).

Proposition 1.2.4.

The conditional probability ℙ(⋅∣A) is a probability measure on (Ω,ℱ).

Proof.

We need to show that (i) ℙ⁢(∅∣A)=0, (ii) ℙ⁢(Ω∣A)=1 and (iii) for pairwise disjoint events (An)n∈ℕ in ℱ we have ℙ⁢(⋃nAn∣A)=∑nℙ⁢(An∣A). (i) is clear because A∩∅=∅. Since Ω∩A=A, it also follows

ℙ⁢(Ω∣A)=ℙ⁢(Ω∩A)ℙ⁢(A)=ℙ⁢(A)ℙ⁢(A)=1,

which establishes (ii). For (iii), note that we have A∩⋃nAn=⋃n(An∩A) and since (An)n are pairwise disjoint, the same remains true for (An∩A)n. Thus, σ-additivity of ℙ yields

ℙ⁢(⋃n∈ℕAn∣A)=ℙ⁢(⋃n∈ℕ(An∩A))ℙ⁢(A)=∑n∈ℕℙ⁢(An∩A)ℙ⁢(A)=∑n∈ℕℙ⁢(An∣A).

∎

Theorem 1.2.5 (Bayes’ formula).

Let A,B∈ℱ such that ℙ⁢(A),ℙ⁢(B)>0. Then, for any B∈ℱ,

ℙ⁢(B∣A)⏞posterior=ℙ⁢(A∣B)⁢ℙ⁢(B)⏞priorℙ⁢(A)=ℙ⁢(A∣B)⁢ℙ⁢(B)ℙ⁢(A∣B)⁢ℙ⁢(B)+ℙ⁢(A∣Bc)⁢ℙ⁢(Bc).
Proof.

We have ℙ⁢(A∣B)=ℙ⁢(A∩B)/ℙ⁢(B), or equivalently, ℙ⁢(A∩B)=ℙ⁢(A∣B)⁢ℙ⁢(B). Plugging this expression for ℙ⁢(A∩B) into ℙ⁢(A∣B)=ℙ⁢(A∩B)/ℙ⁢(B) gives the first stated equality. The second one follows from the first after decomposing A=(A∩B)⊎(A∩Bc), which gives

ℙ⁢(A) =ℙ⁢(A∩B)+ℙ⁢(A∩Bc) (law of total probability)
=ℙ⁢(A∣B)⁢ℙ⁢(B)+ℙ⁢(A∣Bc)⁢ℙ⁢(Bc) (partition formula)

∎

Example 1.2.6.

A doping test is performed after a football match. The league is just a little bit crooked this year so that 0.5% of the players are doped. Moreover, the test detects a doped player with 95% accuracy, but is also positive in 1% of the cases when a non-doped player is tested (false positive). Let us calculate the probability that a player has actually doped if the test returns positive: if we let A=“test positive” and B=“player is doped”, then ℙ⁢(B)=0.005, ℙ⁢(A∣B)=0.95 and ℙ⁢(A∣Bc)=0.01. Thus, using Bayes’ formula

ℙ⁢(B∣A⏟“player is doped given that test is positive”)=ℙ⁢(A∣B)⁢ℙ⁢(B)ℙ⁢(A∣B)⁢ℙ⁢(B)+ℙ⁢(A∣Bc)⁢ℙ⁢(Bc)=0.95⋅0.0050.95⋅0.005+0.01⋅(1−0.005)=0.952.94≈0.32.

This is why in sports there is always a B-sample taken.

Chapter 2 Random variables

2.1 Random variables and their distributions

2.1.1 Measurability

Definition 2.1.1.

Let (E,𝒜) be a measurable space. A function f:E→ℝ is called measurable if for any A∈ℬ⁢(ℝ) it holds that f−1⁢(A)∈𝒜.

This definition appears rather cumbersome at first sight, since σ-algebras such as ℬ⁢(ℝ) are too complicated to check the measurability condition by hand in most circumstances. Fortunately, we can reduce this often impossible task by just checking the preimage condition for generating sets of the Borel σ-algebra, that is open sets or – even simpler – families of intervals as established in Lemma 1.1.7. The formal statement is the following.

Proposition 2.1.2.

Let (E,𝒜) be a measurable space and let ℰ be a generator of ℬ⁢(ℝ), i.e., ℬ⁢(ℝ)=σ⁢(ℰ). Then, a function f:E→ℝ is measurable if, and only if, f−1⁢(ℰ)⊂𝒜.

Proof.

We skip the proof in the lecture. The proof is however quite enlightening, as it contains a powerful idea that is commonly applied in measure theory. I highly recommend to have a look at it in Appendix A. ∎

Example 2.1.3.
  1. (i)

    Let (E,𝒜) be a measurable space and define the indicator function of a set A∈𝒜 as

    𝟏A:E→ℝ,x↦{1,x∈A,0,x∉A.

    For any set B∈ℬ⁢(ℝ) we have

    𝟏A−1⁢(B)={x∈E:𝟏A⁢(x)∈B}={A,1∈B,0∉B,Ac,0∈B,1∉B,E,0,1∈B∅,0,1∉B,

    which shows 𝟏A−1⁢(B)∈𝒜 ⟹ 𝟏A is measurable.

  2. (ii)

    Any continuous function f:ℝ→ℝ is measurable: Let O⊂ℝ be open. By continuity f−1⁢(O) is open as well and since ℬ⁢(ℝ) is generated by open sets, this shows that f−1⁢(O)∈ℬ⁢(ℝ). By Proposition 2.1.2 this is sufficient to prove measurability.

Measurable functions are well-behaved under the usual analytic manipulations. We just record here a couple of important properties properties without proof (if you would like to try to prove some of these, Proposition 2.1.2 is very helpful).

Lemma 2.1.4.

Let (E,𝒜) be a measurable space.

  1. (i)

    If f,g are measurable functions on (E,𝒜), then f+g and f⋅g are also measurable.

  2. (ii)

    If f:E→ℝ and g:ℝ→ℝ are measurable, so is there composition g∘f.

  3. (iii)

    Let (fn)n∈ℕ be a sequence of measurable functions mapping from E to ℝ such that there exists a function f:E→ℝ for which the pointwise convergence limn→∞fn⁢(x)=f⁢(x) holds for any x∈E. Then, f is measurable.

2.1.2 Definition of random variables and their distributions

Definition 2.1.5.

A random variable on a probability space (Ω,ℱ,ℙ) is a measurable mapping X:Ω→ℝ, i.e., X−1⁢(A)∈ℱ for any A∈ℬ⁢(ℝ).

Remark 2.1.6.

In the following we will use the notation {X∈A}≔X−1⁢(A), which is motivated from the definition of the preimage X−1⁢(A)={ω∈Ω:X⁢(ω)∈A}. A random variable is therefore a mapping from the sample space Ω into the real numbers such that for any measurable set A in ℝ the preimage {X∈A} is a proper event on the probability space.

Since random variables are nothing else but measurable mappings in the context of probability spaces, we may recast Lemma 2.1.4 probabilistically.

Lemma 2.1.7.

Let (Ω,ℱ,ℙ) be a probability space.

  1. (i)

    If X,Y are random variables, then also X+Y and X⋅Y are random variables.

  2. (ii)

    If X is a random variable and g:ℝ→ℝ is measurable, then Y=g⁢(X) is also a random variable.

  3. (iii)

    Let (Xn)n∈ℕ be a sequence of random variables such that there exists X:Ω→ℝ for which the pointwise convergence limn→∞Xn⁢(ω)=X⁢(ω) holds for any ω∈Ω. Then, X is also a random variable.

Definition 2.1.8.

The distribution ℙX:ℬ⁢(ℝ)→[0,1] of a random variable X on a probability space (Ω,ℱ,ℙ) is defined by

ℙX⁢(B)=ℙ⁢(X−1⁢(B)),B∈ℬ⁢(ℝ).
Remark 2.1.9.

Analogously to before, we write interchangeably

ℙX⁢(B)=ℙ⁢(X−1⁢(B))=ℙ⁢({X∈B})=ℙ⁢(X∈B).

For an interval [a,b] we also write ℙ⁢(X∈[a,b])=ℙ⁢(a≤X≤b) with analogous conventions for open and half-open intervals. We also use the shorthand ℙ⁢(X∈(−∞,x])=ℙ⁢(X≤x) and write ℙ⁢(X∈A,X∈B)=ℙ⁢({X∈A}∩{X∈B}), ℙ⁢({X∈A}∣{X∈B})=ℙ⁢(X∈A∣X∈B) for A,B∈ℬ⁢(ℝ).

Theorem 2.1.10.

Let X be a random variable on (Ω,ℱ,ℙ). Its distribution ℙX is a probability measure on (ℝ,ℬ⁢(ℝ)).

Proof.
  1. (i)

    ℙX⁢(∅)=ℙ⁢(X−1⁢(∅))=ℙ⁢(∅)=0.

  2. (ii)

    ℙX⁢(ℝ)=ℙ⁢(X−1⁢(ℝ))=ℙ⁢(Ω)=1.

  3. (iii)

    It is easily checked that for a pairwise disjoint sequence (An)n∈ℕ⊂ℬ⁢(ℝ), the sequence (X−1⁢(An))n∈ℕ is also disjoint and by measurability of X it is a sequence in ℱ. Consequently,

    ℙX⁢(⋃n∈ℕAn) =ℙ⁢(X−1⁢(⋃n∈ℕAn))
    =ℙ⁢(⋃n∈ℕX−1⁢(An))
    =∑n∈ℕℙ⁢(X−1⁢(An))
    =∑n∈ℕℙX⁢(An),

    where we used σ-additivity of ℙ for the third line.

∎

Definition 2.1.11.

A random variable X is called discrete if there exists N∈ℕ∪{∞}, weights p1,…,pN≥0 and points x1,…,xN∈ℝ (both interpreted as sequences if N=∞) such that ∑k=1Npk=1 and

ℙX=∑k=1Npk⁢δxk.

We call

fX⁢(x)=ℙX⁢({x})=ℙ⁢(X=x)=∑k=1Npk⁢𝟏{xk}⁢(x),x∈ℝ,

either the discrete density function or probability mass function (pmf).

Example 2.1.12.

Consider a simple coin toss with probability of heads equal to p∈[0,1]. We may model Ω={H,T},ℱ=2Ω={{H},{T},{H,T},∅} and ℙ⁢({H})=p and then set X⁢(H)=1, X⁢(T)=0. The corresponding distribution ℙX can then be written as

ℙX=p⁢δ0+(1−p)⁢δ1,

and we call ℙX a Bernoulli distribution with success rate p. Alternatively, we can start with ℙX as above and use it as a distributional model for the coin toss experiment with random outcome X directly, without specifying the underlying probability space (Ω,ℱ,ℙ).111Note here that if we start with a distribution ℙX on (ℝ,ℬ⁢(ℝ)), we can always construct a probability space (Ω,ℱ,ℙ) and a random variable X:Ω→ℝ such that ℙX is the distribution of X. Indeed, we may simply set Ω=ℝ,ℱ=ℬ⁢(ℝ),ℙ=ℙX and X⁢(ω)=ω for all ω∈ℝ. This is referred to as the canonical construction. The latter modelling approach is particularly useful in situations, where a random phenomenon is far too complex to allow an explicit construction of the underlying probability space (think of a stock price for example) and is therefore the appropriate one for our statistical purposes.

Definition 2.1.13.

The distribution function FX:ℝ→[0,1] of a random variable X on a probability space (Ω,ℱ,ℙ) is defined by

FX⁢(x)=ℙX⁢((−∞,x])=ℙ⁢(X≤x),x∈ℝ.

Distribution functions play a special role because they are much simpler objects compared to distributions (function vs. measure), but are sufficient to fully characterise the distribution.

Theorem 2.1.16.

Let ℙX and ℙY be two distributions. Then, ℙX=ℙY (that is, ℙX⁢(B)=ℙY⁢(B) for all B∈ℬ(ℝ)) if, and only if, FX=FY (that is, FX⁢(x)=FY⁢(x) for all x∈ℝ).

Proof (Sketch).

Generally, the following can be shown to be true: If ℙ, ℚ are two probability measures with underlying σ-algebra ℱ generated by a ∩-stable family of sets ℰ, i.e., ℱ=σ⁢(ℰ) and A∩B∈ℰ if A,B∈ℰ, then ℙ=ℚ iff ℙ⁢(B)=ℚ⁢(B) for all B∈ℰ. Here, we have ℬ⁢(ℝ)=σ⁢(ℰ) for ℰ={(∞,x]:x∈ℝ}, see Lemma 1.1.7, and ℰ is ∩-stable since (−∞,x]∩(−∞,y]=(−∞,x∧y]. Hence, ℙX=ℙY iff

∀x∈ℝ:ℙX⁢((−∞,x])⏟=FX⁢(x)=ℙY⁢((−∞,x])⏟=FY⁢(x).

∎

Proposition 2.1.17 (Properties of a cdf).

A cdf FX has the following properties

  1. (i)

    x↦FX⁢(x) is non-decreasing;

  2. (ii)

    limx→−∞FX⁢(x)=0 and limx→∞FX⁢(x)=1;

  3. (iii)

    x↦FX⁢(x) is right-continuous with left limits, i.e., for any a∈ℝ, limx↑aF⁢(x) exists and limx↓aFX⁢(x)=F⁢(a).

Proof.
  1. (i)

    For x<y we have (−∞,x]⊂(−∞,y] and hence by Lemma 1.1.12 FX⁢(x)=ℙX⁢((−∞,x])≤ℙX⁢((−∞,y])=F⁢(y).

  2. (ii)

    By the just established monotonicity and boundedness of FX, both limits exist and it is enough to check limn→∞FX⁢(−n)=0 and limn→∞FX⁢(n)=1. We only show the first claim, the second is left as an exercise: ((−∞,−n])n∈ℕ is a decreasing sequence, so continuity of measures from Lemma 1.1.14 gives

    limn→∞FX⁢(−n)=ℙX⁢(⋂n∈ℕ(−∞,−n])=ℙX⁢(∅)=0.
  3. (iii)

    Existence of left limits is a consequence of FX being increasing and bounded. Right-continuity can again be proven with continuity of measures (try it!).

∎

2.2 Continuous distributions and a very short sketch of integration theory

In Definition 2.1.11 we have already encountered discrete distributions that place probability mass on single points only. In undergrad probability modules you will also have encountered continuous distributions, which are defined via the cdf as follows: a random variable X is continuous if its cdf is given by

FX⁢(x)=∫−∞xf⁢(y)⁢dy

for a probability density function f:ℝ→ℝ, which is nice enough for the above integral to be well-defined as a Riemann integral. This gives rise to formulas such as

ℙ⁢(a≤X≤b)=∫abf⁢(y)⁢dy=∫[a,b]f⁢(y)⁢dy,

so that we are tempted to define the corresponding distribution as

ℙX⁢(B)=∫Bf⁢(y)⁢dy,B∈ℬ⁢(ℝ).

However, the meaning of such an integral in the sense of Riemann is unclear if the set B is not as nice as an interval (say B=ℚ). We therefore need a different notion of an integral, which is the Lebesgue integral.

2.2.1 Lebesgue integral

We let (E,𝒜,μ) again be a general space. Integrals are supposed to calculate volumes or areas enclosed by functions. We have already interpreted μ⁢(A) as the volume of a set A∈𝒜, which may be interpreted as well as an integral of the indicator 𝟏A w.r.t. the measure μ. So, given some scalar α≥0 it is natural for the function

f⁢(x)=α⁢𝟏A⁢(x),x∈A,

to assign the scaled volume α⁢μ⁢(A) as an integral, that is we define

∫Ef⁢(x)⁢dμ⁢(x)≔α⁢μ⁢(A).

Moreover, integrals should be linear, so for any simple function of the form

f⁢(x)≔∑k=1nαk⁢𝟏Ak⁢(x)⏟fk⁢(x),x∈E,

for some n∈ℕ, α1,…,αn≥0, and A1,…,An∈𝒜 pairwise disjoint, we define

∫Ef⁢(x)⁢dμ⁢(x)≔∑k=1nαk⁢μ⁢(Ak)=∑k=1n∫Efk⁢(x)⁢dμ⁢(x).

To extend the notion of the Lebesgue integral for simple functions to more general function classes, we use the following fundamental result.

Theorem 2.2.1.

For any non-negative, measurable function f:E→ℝ, there exists an increasing sequence of simple functions (fn)n∈ℕ such that we have the pointwise convergence

∀x∈E:limn→∞fn⁢(x)=f⁢(x).
Proof.

Let

fn⁢(x)≔∑k=0n⁢2n−1k2n⁢𝟏f−1([k2n,k+12n)⏟∈𝒜)⁢(x)+n⁢𝟏f−1⁢([n,∞))⏟∈𝒜⁢(x),x∈ℝ.

Then fn is simple and for any x∈ℝ it clearly holds that fn⁢(x)→f⁢(x) as n→∞. Moreover, if x∈f−1⁢([k⁢2−n,(k+1)⁢2−n)), for some k≤n⁢2n−1, then fn⁢(x)=k⁢2−n≤f⁢(x) and similarly, fn⁢(x)=n≤f⁢(x) for x∈f−1⁢([n,∞)). Thus, fn≤f. Finally, it follows from

[k⁢2−n,(k+1)⁢2−n)=⨄l=2⁢k2⁢k+1[l⁢2−(n+1),(l+1)⁢2−(n+1)),[n,n+1)=⨄l=2⁢(n+1)⁢(2n−1)(n+1)⁢2n+1−1[l⁢2−(n+1),(l+1)⁢2−(n+1))

that fn≤fn+1 as desired. ∎

Remark 2.2.2.

We call a sequence of simple functions as above an approximating sequence.

With this result and the intuitive definition of the Lebesgue integral for simple functions, we now define the general Lebesgue integral for any measurable function by taking limits. To this end, we let f+⁢(x)=max⁡{f⁢(x),0} be the positive part and f−⁢(x)=max⁡{−f⁢(x),0} be the negative part of a function f. Note that f=f+−f− and |f|=f++f−. Moreover, if f is measurable, so are f+ and f− and therefore also |f|.

Definition 2.2.3.

Let f:E→ℝ be a non-negative measurable function and (fn)n∈ℕ be an approximating sequence of simple functions. The Lebesgue integral of f is defined by

∫Ef⁢dμ≡∫Ef⁢(x)⁢dμ⁢(x)≔limn→∞∫Efn⁢(x)⁢dμ⁢(x).

We call a measurable function f:E→ℝ μ-integrable if ∫E|f|⁢dμ<∞ and define the Lebesgue integral of an integrable function by

∫Ef⁢dμ=∫Ef+⁢dμ−∫Ef−⁢dμ.
Remark 2.2.4.
  1. (i)

    It can be shown that if (fn)n and (gn)n are two approximating functions of a non-negative measurable function f, then limn→∞∫Efn⁢dμ=limn→∞∫Egn⁢dμ, so that the Lebesgue integral is well-defined.

  2. (ii)

    We will use the notation ∫Af⁢dμ≔∫Ef⁢𝟏A⁢dμ.

  3. (iii)

    If μ=λ is the Lebesgue measure, we usually write ∫Af⁢(x)⁢dλ⁢(x)=∫Af⁢(x)⁢dx.

Next, we state some important properties of the Lebesgue integral. We omit the proof but the strategy is always the same: first establish the desired statement for simple functions, then show it for non-negative measurable functions by taking limits and conclude for μ-integrable functions by decomposing into positive and negative parts.

Proposition 2.2.5.

Suppose that f,g:E→ℝ are μ-integrable.

  1. (i)

    Linearity: for any α,β∈ℝ, the function α⁢f+β⁢g is μ-integrable and

    ∫E(α⁢f+β⁢g)⁢dμ=α⁢∫Ef⁢dμ+β⁢∫Eg⁢dμ.
  2. (ii)

    Monotonicity: if f≤g μ-a.e., then

    ∫Ef⁢dμ≤∫Eg⁢dμ.
  3. (iii)

    Triangle inequality:

    |∫Ef⁢dμ|≤∫E|f|⁢dμ.
  4. (iv)

    If μ⁢(N)=0 then ∫Nf⁢dμ=0.

In many situations it is desirable to switch the order of integration and taking limits and the next two powerful results identify situations where this is allowed.

Theorem 2.2.6 (Monotone convergence).

Let (fn)n∈ℕ be an increasing sequence of measurable non-negative functions converging μ-a.e.  to a function f:E→ℝ, i.e., 0≤f1≤f2≤⋯⁢fn≤⋯ and limn→∞fn=f, μ-a.e. Then,

limn→∞∫Efn⁢dμ=∫Ef⁢dμ. (2.1)

Theorem 2.2.8 (Dominated convergence).

Let f,f1,f2,… be μ-integrable functions such that

  1. (i)

    limn→∞fn=f μ-a.e. and

  2. (ii)

    there exists some non-negative μ-integrable function g s.t. |fn|≤g μ-a.e. for any n∈ℕ,

Then,

limn→∞∫Efn⁢dμ=∫Ef⁢dμ.

2.2.2 Absolutely continuous distributions

With these preparations we can now finally construct continuous distributions.

Proposition 2.2.9 (Change of measure).

Let μ be a measure on (E,𝒜) and f:E→ℝ be a non-negative measurable function such that ∫Ef⁢dμ=1. Then, the set function ν defined by

ν⁢(B)≔∫Bf⁢dμ,B∈𝒜,

is a probability measure on (E,𝒜).

Proof.
  1. (i)

    ν⁢(∅)=∫∅f⁢dμ=∫f⁢𝟏∅⏟≡0⁢dμ=0.

  2. (ii)

    ν⁢(E)=∫Ef⁢dμ=1 by assumption.

  3. (iii)

    Let (An)n⊂𝒜 be pairwise disjoint. Then ∑n=1N𝟏An=𝟏⋃n=1NAn for all N∈ℕ∪{∞} and 0≤f⁢∑k=1n𝟏Ak≤f⁢∑k=1n+1𝟏Ak≤f. Thus,

    ν⁢(⋃n∈ℕAn) =∫Ef⁢𝟏⋃n∈ℕAn⁢dμ
    =∫Ef⁢limn→∞∑k=1n𝟏Ak⁢d⁢μ
    =limn→∞∫Ef⁢∑k=1n𝟏Ak⁢d⁢μ
    =limn→∞∫E∑k=1nf⁢𝟏Ak⁢d⁢μ
    =limn→∞∑k=1n∫Ef⁢𝟏Ak⁢dμ
    =∑n=1∞∫Ef⁢𝟏An⁢dμ
    =∑n=1∞μ⁢(An),

    where we used monotone convergence for the third and linearity for the fifth equality.

∎

Definition 2.2.10.

Let X be a random variable with distribution ℙX that is given by ℙX⁢(B)=∫Bf⁢dμ for a measure μ on (ℝ,ℬ⁢(ℝ)) and a non-negative measurable function f:ℝ→ℝ such that ∫f⁢dμ=1. Then we say that ℙX is absolutely continuous w.r.t. μ (denoted ℙX≪μ) and call f its μ-density. If μ=λ is the Lebesgue measure, we say that ℙX is absolutely continuous and that X is a continuous random variable. We then call f the probability density function (pdf) of ℙX. Generally, we call any non-negative measurable function f such that ∫ℝf⁢(x)⁢dx=1 a pdf and specify its associated distribution by ℬ⁢(ℝ)∋B↦∫Bf⁢(x)⁢dx.

Remark 2.2.11.

If ℙX≪μ we also say that μ dominates ℙX. The density is often symbolically written as f=d⁢ℙXd⁢μ (because then ℙX⁢(B)=∫Bd⁢ℙXd⁢μ⁢dμ) and called the Radon–Nikodym derivative of ℙX w.r.t. μ.

2.3 Expectation of (transformed) random variables

Introducing the Lebesgue integral in the previous section now also allows us to define the expectation of a random variable. Throughout this section we fix a probability space (Ω,ℱ,ℙ) and implicitly understand all random variables to be defined there upon.

Definition 2.3.1.

Let X be a random variable that is either non-negative or ℙ-integrable. The expectation of X is defined by

𝔼⁢[X]=∫ΩX⁢dP.

Since the expectation is just a Lebesgue integral w.r.t. ℙ, we can recast Proposition 2.2.5 probabilistically.

Corollary 2.3.2.

Suppose that X,Y are ℙ-integrable random variables.

  1. (i)

    Linearity: for any α,β∈ℝ, α⁢X+β⁢Y is ℙ-integrable and 𝔼⁢[α⁢X+β⁢Y]=α⁢𝔼⁢[X]+β⁢𝔼⁢[Y].

  2. (ii)

    Monotonicity: if X≤Y, ℙ-a.s., then

    𝔼⁢[X]≤𝔼⁢[Y].
  3. (iii)

    Triangle inequality:

    |𝔼⁢[X]|≤𝔼⁢[|X|].
  4. (iv)

    If X=Y ℙ-a.s., then 𝔼⁢[X]=𝔼⁢[Y].

Recall that if g:ℝ→ℝ is measurable, Y=g⁢(X) is also a random variable, so that the expectation 𝔼⁢[g⁢(X)] is well defined provided g is non-negative or ∫Ω|g⁢(X)|⁢dℙ<∞. To see how to calculate an expected value, let us first look at the case g=𝟏A for some A∈ℬ⁢(ℝ). Then,

𝔼⁢[g⁢(X)]=𝔼⁢[𝟏A⁢(X)]=∫Ω𝟏A⁢(X)⁢dℙ=∫Ω𝟏{X∈A}⁢dℙ=ℙ⁢(X∈A)=ℙX⁢(A)=∫ℝ𝟏A⁢dℙX=∫ℝg⁢dℙX, (2.2)

by the definition of the integral for simple functions. As it turns out, this result is generally true.

Theorem 2.3.3 (Transformation theorem for random variables).

Let X be a random variable and g:ℝ→ℝ be a measurable function that is either non-negative or such that ∫ℝ|g|⁢dℙX<∞. Then,

𝔼⁢[g⁢(X)]=∫ℝg⁢dℙX.
Proof.

Consider first the case that g=∑k=1nαk⁢𝟏Ak is a simple function. Then, using (2.2) and linearity it follows

∫ℝg⁢dℙX=∑k=1nαk⁢∫ℝ𝟏Ak⁢dℙX=∑k=1nαk⁢𝔼⁢[𝟏Ak⁢(X)]=𝔼⁢[∑k=1nαk⁢𝟏Ak⁢(X)]=𝔼⁢[g⁢(X)].

Next, let g≥0 be measurable and let (gn)n∈ℕ be an approximating sequence of simple functions (recall: increasing and converges pointwise to g. Then also (gn⁢(X))n∈ℕ increasing sequence of measurable functions on Ω and limn→∞gn⁢(X)=g⁢(X)). From above, 𝔼⁢[gn⁢(X)]=∫ℝgn⁢dℙX and therefore, applying monotone convergence twice

∫ℝg⁢dℙX=∫ℝlimn→∞gn⁢d⁢ℙX=limn→∞∫ℝgn⁢dℙX=limn→∞𝔼⁢[gn⁢(X)]=𝔼⁢[g⁢(X)].

Finally, suppose that g is measurable and ∫ℝ|g|⁢dℙX<∞. We have just shown that

∫ℝ|g|⁢dℙX=∫Ω|g⁢(X)|⁢dℙ,

so that g⁢(X) is ℙ-integrable. Consequently, the above yields

∫ℝg⁢dℙX=∫ℝg+⁢dℙX−∫ℝg−⁢dℙX=𝔼⁢[g+⁢(X)]−𝔼⁢[g−⁢(X)]=𝔼⁢[g⁢(X)]

as claimed. ∎

As a consequence, the expectations of transformations of a random variable X do not depend on the underlying probability space (Ω,ℱ,ℙ), but can be entirely expressed in terms of the distribution ℙX. This is particularly handy for modelling and explains why in statistics we wish to draw inference on ℙX for a given sample of random variables following this distribution. The above observation also motivates the following definition.

Definition 2.3.4.
  1. (i)

    Let X and Y be two random variables (not necessarily defined on the same probability space). We say that they are equal in distribution if ℙX=ℙY and denote this by X⁢=𝑑⁢Y.

  2. (ii)

    If ν is some distribution, we write X∼ν if ℙX=ν. If ℙX is absolutely continuous with density f, we simply write X∼f.

Remark 2.3.5.

Since distribution functions uniquely characterise the distribution, we have X⁢=𝑑⁢Y iff FX=FY.

2.4 Moments and moment generating functions

We finish this chapter with some important transformations that are particularly important in statistics.

Definition 2.4.1.

The variance Var⁡(X) of a random variable X such that 𝔼⁢[|X|2]<∞ is defined by

Var⁡(X)≔𝔼⁢[(X−𝔼⁢[X])2].

The standard deviation is defined by

σX≔Var⁡(X).

We first note that the variance is well-defined because 𝔼⁢[|X|2]<∞ implies 𝔼⁢[|X|]<∞. This follows from the Cauchy–Schwarz inequality for random variables.

Theorem 2.4.2.

Let X,Y be two random variables on (Ω,ℱ,ℙ). Then,

𝔼⁢[|X⁢Y|]≤(𝔼⁢[X2])1/2⁢(𝔼⁢[Y2])1/2. (Cauchy–Schwarz inequality)

More generally, it holds for any p,q≥1 s.t. 1/p+1/q=1 that

𝔼⁢[|X⁢Y|]≤(𝔼⁢[Xp])1/p⁢(𝔼⁢[Yq])1/q. (Hölder inequality)
Lemma 2.4.3.

Suppose that 𝔼⁢[|X|2]<∞. Then,

Var⁡(X)=𝔼⁢[X2]−𝔼⁢[X]2.
Proof.

By integrability we can use linearity of the expectation and calculate as follows:

𝔼⁢[(X−𝔼⁢[X])2]=𝔼⁢[X2−2⁢X⁢𝔼⁢[X]−𝔼⁢[X]2]=𝔼⁢[X2]−2⁢𝔼⁢[X⁢𝔼⁢[X]]⏟=𝔼⁢[X]⁢𝔼⁢[X]+𝔼⁢[𝔼⁢[X]2]⏟=𝔼⁢[X]2=𝔼⁢[X2]−𝔼⁢[X]2.

∎

Exercise 2.4.4.

The variance is a quantity that roughly encodes how much a random variable X fluctuates around its mean 𝔼⁢[X]. To underline this point we need a fundamental integral inequality that in spite of its simplicity is of fundamental value in statistics.

Theorem 2.4.5 (Markov’s inequality).

Proof.

∀ω∈Ω:|X⁢(ω)|≥t⟹φ⁢(|X⁢(ω)|)≥φ⁢(t)

and therefore

{|X|≥t}⊂{φ⁢(|X|)≥φ⁢(t)}. (2.3)

2.5 P-values and quantiles

Here we get to know two important summary statistics of a random variable X that will be of central importance in our discussion of hypothesis testing later in the lecture. The situation is the following: suppose we are given a test statistic X and we want to check whether the hypothesis X∼P0 is plausible for some given distribution P0 with associated cdf F0. There are different ways to approach this problem:

  1. 1.

    Check whether the observed realization X⁢(ω) is suspiciously small by calculating P0⁢((−∞,X⁢(ω)])=F0⁢(X).

  2. 2.

    Check whether the observed realization X⁢(ω) is suspiciously large by calculating P0⁢([X⁢(ω),∞))=1−F0⁢(X⁢(ω)−).

  3. 3.

    Check both simultaneously by calculating 2⁢min⁡{F0⁢(X⁢(ω)),1−F0⁢(X⁢(ω)−)}.

This motivates the definition of the p-value.

Definition 2.5.1 (p-value).

Suppose that X is a random variable on a probability space (Ω,ℱ,ℙ0) such that X∼P0 under ℙ0.

  1. (i)

    The left-sided p-value pl is defined by pl≔F0⁢(X).

  2. (ii)

    The right-sided p-value pr is defined by pr≔1−F0⁢(X−).

  3. (iii)

    The two-sided p-value ps is defined by ps≔2min{F0(X,1−F0(X−)}.

A small p-value always indicates that the hypothesis X∼P0 might not be true. More precisely:

Lemma 2.5.2.

Suppose that X∼P0 under ℙ0. Then, the following are true for any α∈(0,1):

(i)ℙ0⁢(F0⁢(X)≤α)(ii)ℙ0⁢(1−F0⁢(X−)≤α)(iii)ℙ0⁢(2⁢min⁡{F0⁢(X),1−F0⁢(X−)}≤α)}⁢≤α,

with equality if X is continuous.

For the proof we introduce the pseudo-inverse of a distribution function F0 as

q0:(0,1)→ℝ,q0⁢(α)≔inf{x∈ℝ:F0⁢(x)>α}.
Proof.

For simplicity, we only consider the case that X is continuous. From Exercise 2.2.12 we know that then F0 is a continuous function and we claim that

F0⁢(q0⁢(α))=α. (2.5)

Indeed, observe that for any x<q0⁢(α) we have F0⁢(x)≤α, and therefore by continuity of F0,

F0⁢(q0⁢(α))=limx↑q0⁢(α)F⁢(x)≤α.

On the other hand, for x>q0⁢(α) it holds that F0⁢(x)>α and hence by right-continuity

F0⁢(q0⁢(α))=limx↓q0⁢(α)F⁢(x)≥α.

Combining both statements gives (2.5). Let us also show that

x≤q0⁢(α)⇔F0⁢(x)≤F0⁢(q0⁢(α)). (2.6)

Direction ⟹ is clear since F0 is increasing. Conversely, if F0⁢(x)≤F⁢(q0⁢(α)), then by (2.5), we have F0⁢(x)≤α, whence x≤q0⁢(α). Thus,

ℙ0(F0(X)≤α)=(⁢2.5⁢)ℙ0(F0(X)≤F0(q0(α))=(⁢2.6⁢)ℙ0(X≤q0(α))=F0(q0(α))=(⁢2.5⁢)α.

The remaining statements are easily derived from the first (try this!). ∎

Definition 2.5.3 (γ-quantile).

Let γ∈(0,1). A real number qγ is called γ-quantile of a distribution P, if

P⁢((−∞,qγ])≥γandP⁢([qγ,∞))≥1−γ.
Remark 2.5.4.
  1. (i)

    Interpretation: qγ splits the real axis in left and right half-lines with probability at least γ and 1−γ, respectively.

  2. (ii)

    Equivalently, qγ is a γ-quantile iff F⁢(qγ−)≤γ≤F⁢(qγ).

  3. (iii)

    If F0 is strictly increasing, γ-quantiles are unique.

3.1 Random vectors

Definition 3.1.1.

Every n-dimensional random vector is an ℝn-valued random variable and vice versa.

The joint distribution of a random vector is a multivariate distribution.

Theorem 3.1.11.

𝔼⁢[g⁢(X1,…,Xn)]=∫ℝng⁢(x1,…,xn)⁢dℙX1⊗⋯⊗Xn⁢(x1,…,xn).

3.1.1 Product measures and Fubini’s theorem

Calculating such expectations can be very challenging, unless the joint distribution has a fairly nice structure. The essential tool in such situations is Fubini’s theorem. This, however, first requires the understanding of a product measure, which we will also heavily encounter in the next section on independence. Let (E1,𝒜1,μ1) and (E2,𝒜2,μ2) be σ-finite measure spaces. The product σ-algebra is defined as

𝒜1⊗𝒜2≔σ⁢(𝒜1×𝒜2)=σ⁢({A1×A2:A1∈𝒜1,A2∈𝒜2}),

and the product measure μ1⊗μ2 is the unique measure on 𝒜1⊗𝒜2 that satisfies

∀A1×A2∈𝒜1×𝒜2:μ1⊗μ2(A1×A2)=μ1(A1)μ2(A2).

Observe here that we have already encountered such a product measure before: the multivariate Lebesgue measure λn.

Because the order of integration does not matter, in short we may therefore just write

3.2 Independence

The standard assumption in statistics is the following: we are given n random samples X1,…,Xn that are independent and follow a common distribution P that we wish to infer from the data. The definition of independence of random variables naturally derives from independence of events that we discussed in Chapter 1.

Definition 3.2.1 (Independence of random variables).
  1. (i)

    Let X1,…,Xn be random variables, all defined on the same probability space (Ω,ℱ,ℙ). We say that X1,…,Xn are independent if

    ∀B1,…,Bn∈ℬ(ℝ):ℙ(X1∈B1,…,Xn∈Bn)=∏i=1nℙ(Xi∈Bi).

    If additionally ℙX1=⋯=ℙXn we say that X1,…,Xn are i.i.d. (independent and identically distributed). We then also call X1,…,Xn an i.i.d. sample

  2. (ii)

    A sequence of random variables X1,X2,… on a common probability space (Ω,ℱ,ℙ) is defined to be independent (resp., i.i.d) if for any n∈ℕ the random variables X1,…,Xn are independent (resp., i.i.d.).

In terms of the joint distribution and the product distribution, the definition of independence reads equivalently

ℙX1⊗⋯⊗Xn(B)=ℙX1⊗⋯⊗ℙXn(B),∀B∈×i=1nℬ(ℝ).

Since ×i=1nℬ⁢(ℝ) is a ∩-stable generator of ℬ⁢(ℝn) and probability measures are uniquely characterised on such, we obtain the following nice formula.

Proposition 3.2.2.

A family of random variables X1,…,Xn is independent if, and only if,

ℙX1⊗⋯⊗Xn=ℙX1⊗⋯⊗ℙXn.

Based on this we collect other useful characterisations of independence.

Corollary 3.2.3.

A family of random variables X1,…,Xn is independent if, and only if, for every collection g1,…,gn:ℝ→ℝ of measurable, non-negative functions it holds that

𝔼⁢[∏i=1ngi⁢(Xi)]=∏i=1n𝔼⁢[gi⁢(Xi)]. (3.1)
Proof.

Suppose that X1,…,Xn are independent. Then, using Theorem 3.1.11, Proposition 3.2.2 and Fubini’s theorem, we have

𝔼⁢[∏i=1ngi⁢(Xi)] =∫ℝng1⁢(x1)⁢⋯⁢gn⁢(xn)⁢dℙX1⊗⋯⊗ℙXn⁢(x1,…,xn)
=∫ℝn−1(∫ℝg1⁢(x1)⁢∏i=2ngi⁢(xi)⁢d⁢ℙX1⁢(x1))⁢dℙX2⊗⋯⊗ℙXn⁢(x2,…,xn)
=lin.⁢∫ℝn−1∏i=2ngi⁢(xi)⁢(∫ℝg1⁢(x1)⁢dℙX1⁢(x1))⁢d⁢ℙX2⊗⋯⊗ℙXn⁢(x2,…,xn)
=lin.⁢∫ℝg1⁢(x1)⁢dℙX1⁢(x1)⁢∫ℝn−1∏i=2ngi⁢(xi)⁢d⁢ℙX2⊗⋯⊗ℙXn⁢(x2,…,xn).

3.2.1 Multivariate normal distribution

Definition 3.2.12.

Let μ∈ℝn and Σ∈ℝn×n be a symmetric positive semi-definite matrix. A random vector X=(X1,…,Xn)⊤ is defined to have a multivariate normal (or Gaussian) distribution with mean vector μ=𝔼⁢[X] and covariance matrix Σ, if for any real numbers a1,…,an,

a⊤⁢X=∑i=1ai⁢Xi∼𝒩⁢(a⊤⁢μ,a⊤⁢Σ⁢a)=𝒩⁢(∑i=1nai⁢μi,∑i,j=1nai⁢aj⁢Σi,j).

It is immediate from this definition that every entry Xi of a multivariate normal vector is normally distributed as well with Xi∼𝒩⁢(μi,Σi,i). It is important to remember however that if we are given Xi∼𝒩⁢(μ,σi2), i=1,…,n, then (X1,…,Xn)⊤ is not necessarily a multivariate random vector. If Σ=𝕀n is the identity, i.e., Cov⁡(Xi,Xj)=𝟏{i}⁢(j), X is called a standard normal random vector.

Multivariate normal distributions can always be generated by linearly transforming a standard normal random vector. Let Σ1/2∈ℝn×n denote the symmetric square root of a covariance matrix Σ, that is Σ=Σ1/2⁢Σ1/2=Σ1/2⁢(Σ1/2)⊤ (the square root always exists because we have seen that covariance matrices are symmetric positive semi-definite).

Theorem 3.2.13.

X∼𝒩⁢(μ,Σ) if, and only if, X⁢=𝑑⁢μ+Σ1/2⁢Z for a standard normal random vector Z.

For the proof we use the so-called Cramer–Wold device, which can be proven via multivariate characteristic functions (not part of this module).

Theorem 3.2.14 (Cramer–Wold device).

For two n-dimensional random vectors X and Y it holds that

X⁢=𝑑⁢Y⇔∀a∈ℝn:a⊤⁢X⁢=𝑑⁢a⊤⁢Y.
Proof of Theorem 3.2.13.

By the Cramer–Wold device it suffices to check that a⊤⁢X⁢=𝑑⁢a⊤⁢(μ+Σ1/2⁢Z) for any a∈ℝn. By definition of X and Z it holds that a⊤⁢X∼𝒩⁢(a⊤⁢μ,a⊤⁢Σ⁢a) and

a⊤⁢(μ+Σ1/2⁢Z)=a⊤⁢μ+a⊤⁢Σ1/2⁢Z⏟∼𝒩⁢(0,a⊤⁢Σ1/2⁢(a⊤⁢Σ1/2)⊤)=𝒩⁢(0,a⊤⁢Σ1/2⁢(Σ1/2)⊤⁢a)=𝒩⁢(0,a⊤⁢Σ⁢a)∼𝒩⁢(a⊤⁢μ,a⊤⁢Σ⁢a),

whence a⊤⁢X⁢=𝑑⁢a⊤⁢(μ+Σ1/2⁢Z). ∎

If X∼𝒩⁢(μ,Σ), then Cov⁡(Xi,Xj)=Σi,j.

Gamma distribution

We say that a continuous random variable X has a Gamma distribution with shape parameter α and scale parameter β>0, if its density is given by

fα,β⁢(x)=βαΓ⁢(α)⁢xα−1⁢e−β⁢x⁢𝟏(0,∞)⁢(x),

where

Γ⁢(z)≔∫(0,∞)xz−1⁢e−x⁢dx,z>0,

is the Gamma function, which is easily shown to satisfy Γ⁢(1)=1 and the recurrence relation

Γ⁢(z+1)=z⁢Γ⁢(z),z>0.

Moreover, for later use we note that Γ⁢(1/2)=π since substituting x=y2 (↝dx=2ydy) yields

Γ⁢(1/2)=∫(0,∞)x−1/2⁢e−x⁢dx=∫0∞1y⁢e−y2⁢2⁢y⁢dy=2⁢∫0∞e−y2⁢dy=∫ℝe−y2⁢dy=π.

As it turns out, for α∈ℕ the Gamma distribution is the distribution of the sum of i.i.d. exponential random variables. 3.3 Conditional distributions and expectations

We now introduce conditional distributions and expectations. The general measure theoretic route would go via specifying conditional expectations first and then define conditional distributions from there.
